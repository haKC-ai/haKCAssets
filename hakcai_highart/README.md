# haKCai Animated Logos

Animated transparent-background GIFs of the haKCai ASCII logo, rendered using every [terminaltexteffects](https://pypi.org/project/terminaltexteffects/) effect via [hakcer](https://pypi.org/project/hakcer/).

**29 effects** · **40 frames each** · **60ms/frame** · **Transparent RGBA background**

Click any animation to open the full GIF.
